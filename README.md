# Real Estate Market ZP

## Tech requirements
 - Rails 5.1
 - Ruby 2.4.1
 - SCSS
 - ES6
 - HAML
 - Bootstrap 3.3.6
 - PG

## To Start project
To start project run

`bundle install`

`rails db:create db:migrate db:seed`

`rails s`

To run test `rspec spec`
